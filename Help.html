F.A.Q.s

Q1: Im having connection issues, what should I do?

If you're experiencing connection issues or encountering errors while trying to join the server, please check your internet connection, Minecraft version compatibility, and server status. If the problem persists, feel free to contact our support team for assistance.

Q2: What is a server IP address?

The server IP address is a numerical label assigned to each Minecraft server. It serves as the unique identifier for the server on the internet, allowing players to connect to the server using its IP address.

Q3: What kind of information can I find on this site?

We offer a variety of broader MineCraft information and primarily focus on updating on various MineCraft servers and their status.

Q4: Wny does a server claim to be active but is actually offline

There are many possible reasons, try to refresh the page as old data could be loaded on your webpage. Other reasons could include errors in the code on our end, issues with the server owner's connection or a bad user connection. Always make sure to be connected to the internet when checking and refreshing our site.

Q5: Why is there outdated information on the website?

It is very possible we have made an oversight. If you have not done so already feel free to use our contact form under the "Contact" tab at the top of the screen to let us know and we will check on that information as soon as we can to ensure it is accurate and up to date.

HELPFUL TIPS

  • Check for errors (i.e. spelling errors of server names, etc.
  • Make sure server is not outdated
  • Make sure server status is Active
